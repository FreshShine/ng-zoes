<h1>工具类</h1>


### 说明
| 类名       | 说明           |  使用 |
| ------------- |:-------------:|:-------------: |
| `toBool`     | 返回布尔值 | `toBool(value: any, defaultValue: boolean)` |
| `isImageType` | 判断是否是图片类型 | `isImageType(type)` |
| `isVideoType` | 判断是否是视频类型 | `isVideoType(type)` |
| `getFileType` | 根据文件后缀判断类型 | `getFileType(fileName)` |
